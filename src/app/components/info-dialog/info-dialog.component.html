General information

This project is a front end tool for generating monitoring scripts for linux systems derived from debian.
The nativeshell scripts were chosen so that there is no system disturbance at the time of data collection.
Basic prerequisites:
date, mpstat, free, df, cat, ps aux.
Auxiliary prerequisites:
echo, awk, cut, gawk, grep, head, sleep, wc.
Execution prerequisite:
chmod +x example.sh
Execution:
./example.sh
Instructions for use
First: Assemble the structure with the available nodes.
Second: Configure each node.
Third: Press the run button to generate the scripts.
